Le programme NameSpace2Java génère, à partir du fichier OOM (en XML) de Sybase Power AMC Designer et des fichiers ksp définissants les DTO et les services, les classes avec 
les accesseurs correspondants.
Il faut préciser en argument un fichier properties avec les paramètres. targetDir : repertoire de destination des fichiers à générer. 
kpr : chemin du fichier kpr listant les fichiers ksp (facultatif) oom : chemin du fichier oom (facultatif) javadoc : booléen true ou false indiquant si la javadoc doit être incluse dans les classes générées sequenceFormat : format des séquences (SEQ_{0} par défaut)



Le comportement lors de la génération des méthodes liées aux services est le suivant:
Les services peuvent avoir un nombre illimité de paramètres in quel que soit leur type.
Les services peuvent avoir un nombre illimité de paramètres inout mais ceux-ci ne peuvent pas être de type primitif.

Les services peuvent avoir un seul paramètre out qui donne lieu à une méthode retrieve dans tous les cas.
